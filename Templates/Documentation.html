Documentation

Cette API permet de scraper des données sur Alibaba, de les nettoyer et de les stocker dans une base PostgreSQL. Elle est containerisée avec Docker et surveillée avec Prometheus et Grafana.

Technologies utilisées

FastAPI

Pour l'API

Selenium

Pour le scraping

PostgreSQL

Base de données

Celery

Gestion des tâches asynchrones

Docker

Pour la conteneurisation

Prometheus & Grafana

Pour le monitoring
